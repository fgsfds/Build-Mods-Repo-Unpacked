==========================================================================

Version:         1.1
Release Date:    22.04.2020
Author:          fgsfds (https://twitter.com/fgsfds228)


==========================================================================
ABOUT
==========================================================================

  This pack replaces tiles that were changed in the World Tour release
with the original ones. This includes red cross logos, credit cards,
and some info screens.


==========================================================================
HOW TO USE
==========================================================================

BuildGDX (1.08 and later) and EDuke32:
- put wtuncensor.zip into autoload folder
- make sure "enable autoload folder" checkbox is enabled
