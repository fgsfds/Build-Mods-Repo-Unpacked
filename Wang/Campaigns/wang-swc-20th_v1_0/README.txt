Shadow Warrior Central 20th 
--------------------------
Released: October 26 2016
Created By: Robman.

Released as Freeware. 

Best played with voxels on, midi music on low, built with difficulty #3 as the default. 

Best played with SWP or newer ports.(Built with SWPbuild)
The HRP and Redux removes the HUD changes. Also, if the game is too dark, don't be afraid to adjust the brightness. (F11 in SWP)

---------------------------------------------

Maps 1-4 Multiplayer only.
1- The Edge (MP)
2- Danzig (MP)
3- ProArena (MP)
4- Eagle (MP)

Maps 5-15 Singleplayer (most are multiplayer capable also) - Midi Music
5- StarGate (SP) - Know your enemy
6- StarGate Continued... (SP/MP) - Enter Sandman
7- Tron (SP/MP)
8- Shootout (SP) - Thunderstruck
9- LoTrek (SP/MP) - Star Trek Theme (This map is originally by "Wamplet" - I've modified it extensively)
10- WangBusters (SP/MP) - GhostBusters Theme
11- Shadow (SP/MP) - Praise You
12- Heretic (SP/MP) - Heretic E1M1 Music
13- Matrix (SP) - Wake Up
14- Sw World (SP/MP)

22- Sw Land (Bonus Map) (accessible from secret exit in map 5)

I play it with SWP, voxels on, midi music on low, skies only from the HRP, difficulty #3.
SAVE OFTEN!
