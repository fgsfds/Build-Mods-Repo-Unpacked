======================================================
Black Phantasm Mod for Blood
======================================================

Version:       1.0
Release Date:  26.06.2020


======================================================
ABOUT
======================================================

This pack replaces Phantasm sprites with the black variant.
It is meant to be used with BuildGDX or NBlood ports.

======================================================
HOW TO USE
======================================================

BuildGDX (1.13 or later) and NBlood:
- put blackph.zip into autoload folder in your Blood folder
- make sure "enable autoload folder" checkbox is enabled
