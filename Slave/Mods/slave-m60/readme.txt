Powerslave/Exhumed Alternate M60 Sprite
============================================================================================

Version:       1.0
Release Date:  01.05.2020
Author:        fgsfds


============================================================================================
ABOUT
============================================================================================

This pack provides replacement for the M60 sprites in Powerslave/Exhumed.
It is meant to be used with the BuildGDX and PCExhumed ports.


============================================================================================
HOW TO USE
============================================================================================

Drop psm60.zip into "autoload" folder.
